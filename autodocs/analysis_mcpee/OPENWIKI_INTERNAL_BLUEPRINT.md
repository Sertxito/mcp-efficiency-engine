# OpenWiki Internal Blueprint

Ubicacion canonica del blueprint y analisis de evolucion de AutoDocs como wiki
interna de `mcp-efficiency-engine`.

Estado actual implementado:

- runtime y artefactos movidos a `autodocs/`
- normalizacion de `wiki_nodes`
- validacion estructural con reportes
- proyeccion humana a `autodocs/site/`
- manifests de busqueda y navegacion
- workflow CI actualizado

Siguiente nivel de madurez:

- ampliar providers mas alla de capacidades
- enriquecer taxonomy por agentes, skills, policies y specs reales
- añadir UI estatica si se quiere una experiencia mas cercana a OpenWiki
